# Docker Laber (WIP)

This project aims to detect all the docker containers running in the local machine and check for new images for updates. 

## Build and test 

 - `cargo build`
 - `cargo test`

## Run 

 - `cargo run` visit ---> http://127.0.0.1:8080/
